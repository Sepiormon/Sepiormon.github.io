Pathfinding Project. This project focused on allowing a character on a 2D grid to find a path on a tilemap, following certain rules for which tiles can or cannot be accesed. Also Allows for visual representation of which times any one character can reach given they have enough movement remaining.

Bomb Defusal Project This Project focuses on the skills required to take an .exe file and disassemble it and interpret its instructions without ever having to run it. Successful individuals will defuse the bomb when it runs, otherwise it will explode on you.

Arcade Game Project Work-In-Progress This project is an attempt at making an arcade game. The game fits the theme and scale of those you would find in a traditional arcade machine from the 90's. Pew pew pew pew pew.
